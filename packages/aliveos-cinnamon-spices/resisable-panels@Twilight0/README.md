# Resisable Panels

A simple, lightweight Cinnamon extension to customize panel sizes (lengths) and margins on horizontal and vertical screen edges.

## Features
*   **Individual Panel Configuration**: Configure each of your active panels independently.
*   **Next/Previous Panel Selectors**: Cycle through all running panels directly inside the configuration dialog.
*   **Active Highlighting**: Highlights the targeted panel with a visible border on your screen to identify which panel is being configured.
*   **Custom Sizing**: Shrink horizontal (top/bottom) panels or vertical (left/right) panels to a custom percentage of screen dimensions.
*   **Floating Panels**: Detach panels from the screen edges (float mode) and customize the exact distance (in pixels) they hover away from the edge.
*   **Edge Alignment**: Resized panels stay perfectly centered on their respective monitor edges.
*   **Lightweight**: Relies on native Cinnamon panel styles and layout changes without heavy window-tracking event loops.

## Requirements
*   Cinnamon 5.0 or newer.

## Installation
Place the `resisable-panels@Twilight0` directory under `~/.local/share/cinnamon/extensions/` and enable it in Cinnamon's Extensions settings.
