Quick Setup

  1. Add the repository to /etc/pacman.conf
  2. Sync and install packages
[aliveos-repo]
SigLevel = Optional TrustAll
Server = https://github.com/Twilight0/aliveos-repo/releases/download/latest

Then run:

sudo pacman -Syu
sudo pacman -S <package-name>
